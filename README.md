# 📊 Exercícios de Vetores em Java

Repositório dedicado à prática de vetores (arrays) em Java, com foco no desenvolvimento da lógica de programação e
manipulação de dados em memória.

---

## 🎯 Objetivo

Desenvolver habilidades fundamentais em programação utilizando estruturas de dados simples, como vetores,
reforçando conceitos essenciais para evolução no desenvolvimento backend.

---

## 🧠 Conteúdos abordados

* Declaração e inicialização de vetores
* Percorrendo arrays com estruturas de repetição (`for`, `while`)
* Leitura de dados pelo usuário
* Operações com vetores:

  * Soma de elementos
  * Média
  * Maior e menor valor
* Ordenação básica
* Busca de elementos

---

## 💻 Tecnologias utilizadas

* Java
* IDE (IntelliJ / Eclipse)
* Console (entrada e saída de dados)

---

## 📌 Exemplos de exercícios

* 🔹 Ler 10 números e exibir a média
* 🔹 Encontrar o maior e menor valor de um vetor
* 🔹 Contar números pares e ímpares
* 🔹 Buscar um valor específico no vetor
* 🔹 Ordenar os elementos
